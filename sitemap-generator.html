🗺️ Sitemap Generator

Internal tool — generates sitemap.xml from every approved business currently in Firestore, plus all static pages. This is not a public-facing page; it produces the file content you upload to your server as /sitemap.xml.

⚠️ Static hosting limitation: Since this platform runs as static HTML + client-side Firestore (no server/Cloud Functions), the live sitemap.xml file on your server cannot regenerate itself automatically on a schedule. Two ways to keep it fresh:

1. Manual refresh (works today, zero cost): Open this page periodically (e.g. weekly, or after a batch of new listings), click "Generate", download the file, and re-upload it to replace /sitemap.xml on your host.

2. Automated (recommended once volume grows): Add one small Firebase Cloud Function on a daily schedule (functions.pubsub.schedule) that runs this exact same logic server-side and writes the output to Cloud Storage or back into your static hosting bucket. This is the only piece of the whole platform that genuinely benefits from a Cloud Function — everything else can stay client-side.
